Recall that exponents (also known as powers) are used to denote a number multiplied by itself many times. We can extend the idea of exponents to negative numbers and fractions as well.

Let's try to evaluate an exponent of a negative number, \((-4)^3\).
\((-4)^3 = \scriptsize - \normalsize 4 \times \scriptsize - \normalsize 4 \times \scriptsize - \normalsize 4 = -64\)

Exercise 1

Evaluate \((-9)^2\)


Show solution
The correct answer is \( 81 \).

\((-9)^2 = \scriptsize - \normalsize 9 \times \scriptsize - \normalsize 9 = 81\).


There is a neat trick that we can use to determine the sign of the answer (in other words, if the answer is positive or negative) when calculating exponents of negative numbers: If the exponent is an odd number (in other words, the exponent is not divisible by 2), the result is a negative number. If the exponent is an even number (in other words, the exponent is divisible by 2 the result is a positive number. Note that this trick only applies when the base is a negative number. If the base is a positive number, the result will always be positive.

Exercise 2

Is the result of \((-157)^9\) a positive number, a negative number, or 0?


Show solution
The result of \((-157)^9\) is a negative number.

Because the base -157 is negative and the exponent 9 is odd, the result is a negative number.


Exercise 3

Is the result of \(157^9\) a positive number, a negative number, or 0?


Show solution
The result of \(157^9\) is a positive number.

Because the base 157 is positive, the result will be positive.


Calculating exponents of fractions is also easy to do. For example, let's try to calculate \((\frac{1}{2})^3\). \((\frac{1}{2})^3 = \frac{1}{2} \times \frac{1}{2} \times \frac{1}{2} = \frac{1 \times 1 \times 1}{2 \times 2 \times 2} = \frac{1}{8}\).

A neat shortcut that we can use when we want to calculate the exponent of a fraction is to distribute the exponent to the numerator and the denominator. For example, let's try to calculate \((\frac{1}{2})^3\) again but using this shortcut this time. \((\frac{1}{2})^3 = \frac{1^3}{2^3} = \frac{1 \times 1 \times 1}{2 \times 2 \times 2} = \frac{1}{8}\).

If we were to compare the two processes, we could see that what we just did is pretty much the same as before, the only difference being the skipping of a step in the middle. Both of these methods will always give the same result, but the second way is slightly faster.

Exercise 4

Evaluate \((\frac{5}{3})^2\).


Show solution
The correct answer is \( \frac{25}{9} \).

Using the first method: \((\frac{5}{3})^2 = \frac{5}{3} \times \frac{5}{3} = \frac{5 \times 5}{3 \times 3} = \frac{25}{9}\)
Using the second method: \((\frac{5}{3})^2 = \frac{5^2}{3^2} = \frac{5 \times 5}{3 \times 3} = \frac{25}{9}\)


Always check if there are parentheses around a fraction when doing exponents! For example, \(\frac{3^2}{2}\) is NOT the same as \((\frac{3}{2})^2\)! In the first case, only the numerator is being multiplied to itself, so we are NOT calculating an exponent of a fraction. In the second case, the whole fraction is being multiplied to itself, so we are calculating an exponent of a fraction.

Exercise 5

Evaluate the three expressions: \((\frac{3}{2})^2\), \(\frac{3^2}{2}\), \(\frac{3}{2^2} \).


Show solution
The correct answers are \((\frac{3}{2})^2 = \frac{9}{4}\), \(\frac{3^2}{2} = \frac{9}{2}\), \(\frac{3}{2^2} = \frac{3}{4}\).

\((\frac{3}{2})^2 = \frac{3^2}{2^2} = \frac{3 \times 3}{2 \times 2} = \frac{9}{4}\)
\(\frac{3^2}{2} = \frac{3 \times 3}{2} = \frac{9}{2}\)
\(\frac{3}{2^2} = \frac{3}{2 \times 2} = \frac{3}{4}\)
Only in the first case are we taking the exponent of a fraction. In the second and third cases, we are not taking the exponent of a fraction, but of the numerator and denominator respectively. Since the shortcut method only works for taking the exponent of a fraction, it does not work in the second and third cases.
