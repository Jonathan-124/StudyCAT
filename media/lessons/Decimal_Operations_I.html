The process of adding and subtracting decimal numbers is very easy to learn, since it is exactly the same as long addition and subtraction with whole numbers, which we already know how to do! If you need a refresher on long addition or subtraction, check out the prerequisite lessons links above.

Recall that for both long addition and long subtraction with only whole numbers, we aligned numbers by their place, then added/subtracted the numbers. When adding/subtracting decimal numbers, the process is exactly the same, with one caveat: when lining up the numbers by their place, we also need to line up the decimal point and keep the decimal point in the same position in the result underneath the line.

Here's an example: calculate \( 27.3 + 1.8 \).
We set the equation up exactly how we would set it up with only whole numbers, with the additional requirement that we include and align the decimal points: $$ \begin{matrix} &2&7.&3 \\ +&&1.&8 \\ \hline \end{matrix} $$ The process of long addition is exactly the same. After showing our work, we can get the following result: $$ \begin{matrix} &&\tiny 1 \\ &2&7.&3 \\ +&&1.&8 \\ \hline &2&9.&1 \end{matrix} $$ Don't forget the decimal point in the result underneath the line!

Here's an example involving subtraction: calculate \(27.8 - 4.9\): $$ \begin{matrix} &2&\cancel{7}6.&8 \\ -&&4.&9 \\ \hline &2&2.&9 \end{matrix} $$ As shown above, the setup and steps taken are exactly the same, with the exception of keeping track of the decimal point.

Show solution

Note that when adding/subtracting decimal numbers, sometimes certain places might be occupied by digits in one number but not the other number. In these cases, still line up all digits by place when setting the equations up for long addition/subtraction, but replace the blank places with zeroes.

Here's an example: calculate \( 17.02 + 1.2345 \).
Because the first number does not have a thousandths or ten-thousandth place, we treat them as zeroes and add the numbers like so: $$ \begin{matrix} &1&7.&0&2&0&0 \\ +&&1.&2&3&4&5 \\ \hline &1&8.&2&5&4&5 \\ \end{matrix} $$
