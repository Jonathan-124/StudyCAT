Consider this question. Which number is bigger, twenty three thousand six hundred forty three or four thousand eight hundred sixty three?
It is difficult to do this comparison in our heads, since it is hard to visualize the numbers directly. Fortunately, we can always follow a few simple steps to figure out which of any two numbers is bigger, even when the numbers are very large.

The process is as follows:

  1. Write down both numbers using digits.
  2. Compare the amount of places/digits the two numbers have.
    • If the two numbers do not have the same amount of places, the number with more places is bigger.
    • If the two numbers have the same amount of places, compare the place value of the left-most place of the two numbers.
      • If these two digits are different, the number with the bigger place value in this place is bigger.
      • If these two digits are the same, repeat this step but compare the place values in the next place to the right.
Let's try to tackle the original example.

Exercise 2

Which number is bigger?


Show solution
The correct answer is 1419.

1027 and 1089 are already written using digits.
Since both numbers have 4 places, we need to go to compare their place values in their left most places (the thousands' place).
The thousands' place of the two numbers has the same place value (1), so we have to compare the place values in the hundreds' place.
Since 4 is larger than 3, 1419 is larger than 1327.


Exercise 3

Choose the correct sign that fills in the blank. $$ 185 \ \square \ 564 $$


Show solution
The correct answer is \( < \).

185 and 564 are already written using digits.
Both numbers have 3 places, so we need to compare the place values of the left-most place (the hundreds' place).
Since 1 is smaller than 5, 185 is less than 564.
Recall that the pointy end of the comparison sign always points to the smaller number. So \( < \) is the correct sign to fill in the box!
