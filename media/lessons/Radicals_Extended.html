The concept of radicals also extends to decimals and fractions.

With decimals, we can pretty much directly apply all we've learned about radicals. For example, let's try to solve \(\sqrt[3]{0.125}\). Since the index of the radical is the odd number 3, we know that there will be exactly one single solution, and it will be have the same sign as the number underneath the radical symbol. \(0.5\) is the only solution to this radical, which we can verify: \(0.5 \times 0.5 \times 0.5 = 0.125\).

Exercise 1

Find all solutions to \(\sqrt{1.21}\)


Show solution
The correct answers are \( 1.1, -1.1 \)

Since the index of the radical is the even number 2 (recall that a radical symbol without an index on the left means that the index is 2), we know that there will be exactly 2 solutions with oppsite signs. We can verify that \(1.1 ^ 2 = 1.21 \) and \((-1.1)^2 = 1.21\), therefore \(\sqrt{1.21} = 1.1, -1.1\) are the only two solutions.


Exercise 2

Find all solutions to \(\sqrt[3]{-0.001}\)


Show solution
The correct answer is \(-0.1 \).

Since the index of the radical is the odd number 3, we know that there will be exactly 1 solution that has the same sign as the number underneath the radical symbol. We can verify that \((-0.1) ^ 3 = -0.001 \), therefore \(\sqrt[3]{-0.001} = -0.1\) is the only solution.


Radicals of fractions are easier to calculate. A neat shortcut that we can use when we want to calculate the radical of a fraction is to distribute the radical to the numerator and the denominator.

For example, \(\sqrt[3]{\frac{1}{8}}\) is the same as \(\frac{\sqrt[3]{1}}{\sqrt[3]{8}}\). Because the index of \(\sqrt[3]{\frac{1}{8}}\) is an odd number 3, there will only be one answer which would have the same sign as \(\frac{1}{8}\). \(\sqrt[3]{\frac{1}{8}} = \frac{\sqrt[3]{1}}{\sqrt[3]{8}} = \frac{1}{2}\).

Exercise 3

Find all solutions to \(\sqrt{\frac{9}{4}}\)


Show solution
The correct answers are \(\frac{3}{2}, -\frac{3}{2}\).

\(\sqrt{\frac{9}{4}} = \frac{\sqrt{9}}{\sqrt{4}} = \frac{3}{2}\).
Because the index is an even number 2, there will be two answers with opposite signs. Therefore \(- \frac{3}{2}\) is also a solution.
We can verify that \((\frac{3}{2})^2 = \frac{3^2}{2^2} = \frac{9}{4}\) is a solution, and \((- \frac{3}{2})^2 = (\frac{-3}{2})^2 = \frac{(-3)^2}{2^2} = \frac{9}{4}\) is also a solution.


Exercise 4

Find all solutions to \(\sqrt[4]{- \frac{1}{16}}\)


Show solution
There are no solutions to \(\sqrt[4]{- \frac{1}{16}}\).

The number underneath the radical symbol, \(- \frac{1}{16}\) is a negative number. Because the index of the radical is 4, an even number, there are no solutions to this radical.


Always check if the radical sign is covering the whole fraction or not! For example, \(\sqrt{\frac{9}{4}}\) is NOT the same as \(\frac{\sqrt{9}}{4}\)! In the first case, we are taking the radical of the whole fraction while in the second case, we are taking the radical of the numerator only.

Exercise 5

Evaluate the following three expressions: \(\sqrt{\frac{9}{4}}\), \(\frac{\sqrt{9}}{4}\), \(\frac{9}{\sqrt{4}}\)


Show solution
\(\sqrt{\frac{9}{4}} = \frac{\sqrt{9}}{\sqrt{4}} = \frac{3}{2}, - \frac{3}{2}\)

\(\frac{\sqrt{9}}{4} = \frac{3}{4}, - \frac{3}{4}\)

\(\frac{9}{\sqrt{4}} = \frac{9}{2}, - \frac{9}{2}\)

Only in the first case we are taking the radical of the entire fraction. In the second case we are taking the radical only of the numerator, and in the third case we are taking the radical only of the denominator.
