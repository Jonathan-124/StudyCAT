Let's put everything we've learned about order of operations, fractions, and decimals together.

When we do math with decimals and fractions, the order of operations still applies. Recall that the order of operations is summarized by the acronym PEMDAS, where each letter stands for the following operation:

  • P: Parentheses (\(()\), also known as brackets)
  • E: Exponents
  • M, D: Multiplication and Division
  • A, S: Addition and Subtraction
For example, if we wanted to calculate \(9 - 1.2 \div 0.6\), we would first divide before subtracting: \(9 - 1.2 \div 0.6 = 9 - 2 = 7\).

Exercise 1

Calculate \((\frac{1}{2})^2 \times (\frac{1}{3} + \frac{1}{4})\)


Show solution
The answer is \(\frac{7}{48}\).

We first compute what is inside the parentheses, then the exponent, then multiply. \((\frac{1}{2})^2 \times (\frac{1}{3} + \frac{1}{4}) = (\frac{1}{2}) \times (\frac{1}{2}) \times (\frac{4}{12} + \frac{3}{12}) = \frac{1}{4} \times \frac{7}{12} = \frac{7}{48}\)

Exercise 2

Calculate \(1 + 0.6^2 \times 2\)


Show solution
The answer is \(1.72\).

We first compute the exponent, multiply, and add. \(1 + 0.6^2 \times 2 = 1 + 0.36 \times 2 = 1 + 0.72 = 1.72\)

When there are both fractions and decimals inside an expression, we can either convert the expression to fractions or everything to decimals before our calculations.

For example, if we wanted to calculate \(1.5 + 2 \times \frac{1}{4}\), we could do it one of two ways. We could either do the calculation using fractions (\(1.5 + 2 \times \frac{1}{4} = \frac{3}{2} + \frac{2}{1} \times \frac{1}{4} = \frac{3}{2} + \frac{2}{4} = \frac{3}{2} + \frac{1}{2} = \frac{4}{2} = 2\)), or do the calculation using decimals (\(1.5 + 2 \times \frac{1}{4} = 1.5 + 2 \times 0.25 = 1.5 + 0.5 = 2\)). The answer will be the same no matter which way we choose to evaluate the expression. as long as the calculations are done correctly.

Exercise 3

Calculate \(\frac{1}{2} + 0.6\) and express your answer as a decimal and as a simplified fraction.


Show solution
The answer is \(1.1\) and \(\frac{11}{10}\).

As a fraction: (\(\frac{1}{2} + 0.6 = \frac{1}{2} + \frac{0.6}{1} = \frac{1}{2} + \frac{1.2}{2} = \frac{2.2}{2} = \frac{2.2 \times 5}{2 \times 5} = \frac{11}{10}\)).
As a decimal: \(\frac{1}{2} + 0.6 = 0.5 + 0.6 = 1.1\). Both answers are correct, as \(\frac{11}{10}\) and \(1.1\) are the same number expressed in two different ways.

Be aware that we must first compute the numerator/denominator of a fraction before proceeding on to other operations.

For example, to calculate \(3 \times \frac{1+2}{4}\), we must first simplify the numerator before proceeding, \(3 \times \frac{1+2}{4} = 3 \times \frac{3}{4} = \frac{3}{1} \times \frac{3}{4} = \frac{9}{4}\). This is because how fractions are the same as division â€” we can consider the expression \(3 \times \frac{1+2}{4}\) the same as \(3 \times (1 + 2) \div 4\). The numerator and denominator of any fraction are in parentheses, and therefore they must be calculated first.

Exercise 4

Calculate \(\frac{7 - 3 \times 2}{2^3 - 6} + 1\) and express your answer as a decimal and as a simplified fraction.


Show solution
The answer is \(1.5\) and \(\frac{3}{2}\).

As a fraction: \(\frac{7 - 3 \times 2}{2^3 - 6} + 1 = \frac{7-6}{8-6} + 1 = \frac{1}{2} + 1 = \frac{1}{2} + \frac{2}{2} = \frac{3}{2}\).
As a decimal: \(\frac{7 - 3 \times 2}{2^3 - 6} + 1 = \frac{7-6}{8-6} + 1 = \frac{1}{2} + 1 = 0.5 + 1 = 1.5\). Both answers are correct, as \(\frac{3}{2}\) and \(1.5\) are the same number expressed in two different ways.

Finally, recall that on a case-by-case basis, sometimes fractions are easier to work with while other times decimals are easier to work with. There is no easy way to tell, but with experience doing problems it will become second nature!

For example, let's try to calculate (\(0.8 \times \frac{1}{3}\). We could either do the calculation using fractions (\(0.8 \times \frac{1}{3} = \frac{0.8}{1} \times \frac{1}{3} = \frac{0.8 \times 1}{1 \times 3} = \frac{0.8}{3} = \frac{0.8 \times 5}{3 \times 5} = \frac{4}{15}\)), or do the calculation using decimals (\(0.8 \times \frac{1}{3} = 0.8 \times 0.3333333333333... = 0.266666666666...\)). In this case, it is much easier to use fractions.
