Although it is easy to add small numbers in our heads, it is much more difficult to add big numbers. For example, when we calculate \( 5+2 \), we can simply get the answer 7. But if we wanted to calculate something like \( 865 + 93\), it is much more difficult to do without a calculator, or pencil and paper.
Long addition is the process we use to add big numbers. Here's the way to do it:

  1. Write one number on top of the other, and match each place vertically. Write a plus sign on the left, and draw a horizontal line underneath the bottom number.
  2. Add all the digits above the line in the ones' place column (on the far right).
    • If the result is a one-digit number:
      • Write the resulting digit underneath the line in the same column.
    • If the result is a two-digit number:
      • Write the digit in the ones' place underneath the same column
      • Write the digit in the tens' place on top of the column to the immediate left.
  3. Repeat from step 2, but now looking at the column to the immediate left of the digit below the line.
  4. When there are no columns left to look at, the number below the line is the final answer!
Let's look at an example.

Calculate \( 865 + 93 \).
First, we set the numbers up by writing the two numbers vertically and matching the digits in step 1: $$ \begin{matrix} &8&6&5 \\ +&&9&3 \\ \hline \end{matrix} $$ Since only the first digit has a hundreds' place, we leave the hundreds' place for the second digit blank. Remember that we match the digits by place, so the 5 must be matched with the 3 and the 6 must be matched with the 9! Note that we can also put the 93 on top and 865 on bottom (which would not change the answer), but we would still have to match the digits by place. In step 2, we start by adding the digits in the ones' place (\( 5+3 \)), and write the result underneath the line like this. $$ \begin{matrix} &8&6&5 \\ +&&9&3 \\ \hline &&&8 \end{matrix} $$ Then, we repeat from step 2, but look at the place immediately to the left and look at digits in the tens' place instead. Because the digits in the tens' place add up to a two-digit number (\( 6+9=15 \)), we write the ones' digit of the result in the same column underneath the line, and the tens' digit of the result on top of the column to the immediate left like this: $$ \begin{matrix} &\scriptsize 1 \\ &8&6&5 \\ +&&9&3 \\ \hline &&5&8 \end{matrix} $$ Finally, we repeat step 2, and look at the next column to the left. Because there is no hundreds' digit in the second number (same as having a 0), we just add \( 1+8 \) and write the result underneath the line in the same column. And because there are no more columns to the left, we are done with the end result looking like this: $$ \begin{matrix} &\scriptsize 1 \\ &8&6&5 \\ +&&9&3 \\ \hline &9&5&8 \end{matrix} $$ The result of \( 865 + 93\) is the number underneath the line, \( 958 \)!
