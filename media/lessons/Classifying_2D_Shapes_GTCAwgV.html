Previously, we have learned that a two-dimensional shape (or 2D shape) is any "flat" shape that does not have any thickness. Let's talk a bit more about how we classify the most common 2D shapes we see everyday.

All 2D shapes are either open or closed.
An open shape has at least one gap, so its sides do not touch. A closed shape has all sides connected to each other.


All closed shapes are either polygons or not polygons.

All polygons can only have edges and vertices.
Edges are straight line segments that make up the boundary or outline of the shape.
Vertices are the points where the edges meet. Each vertex is a "corner" where edges meet.

So in other words, a closed shape drawn with only straight lines is a polygon, and any closed shape that includes non-straight lines is not a polygon!
