The absolute value of a number is its distance from \(0\) on the number line, regardless of whether the number is negative or positive. Here is an example. The number \(3\) has a distance of \(3\) from \(0\) on the number line. Therefore, the absolute value of \(3\) is \(3\).



Here is another example. The number \(-3\) has a distance of \(3\) from \(0\) on the number line. Therefore, the absolute value of \(-3\) is \(3\).


Another way to think about absolute value is that it simply takes away the negative sign of the number inside of it. If a number is positive, when we take its absolute value nothing happens; if a number is negative, when we take its absolute value we just take away the negative sign to get the answer. This is because the number line is symmetrical around 0. The negative "version" of a number always has the same distance to \(0\) as its positive "version".

The absolute value symbol \(| \ \ \ |\) is two vertical lines that surround the number that we want to find the absolute value of. With the absolute value symbol, instead of saying "the absolute value of \(-3\) is \(3\)", we could just write \(|-3| = 3\).

Exercise 1

Evaluate \(|-7|\)


Show solution
The correct answer is \( 7 \).

If we were to draw out a number line, \(-7\) would be a distance of \(7\) away from \(0\). Therefore, \(|-7| = 7\)


Exercise 2

Evaluate the following expressions: \(|1|, |-1|, -|1|, -|-1|\)


Show solution
\(|1| = 1 \\ |-1| = 1 \\ -|1| = -1 \\ -|-1| = -1\)

In the first two cases, the absolute value of \(1\) and \(-1\) are both \(1\).
In the third and fourth cases, we are taking the negative of their absolute values, and the results are therefore both \(-1\).


Exercise 3

Choose the correct sign that fills in the blank. $$ |-4| \ \square \ 2 $$


Show solution
The correct answer is \( > \).

\(|-4| = 4\), and since \(4 > 2\), \(|-4| > 2\)
