The process of dividing decimal numbers is very easy to learn, since it is similar to long division with whole numbers, which we already know how to do! If you need a refresher on long division, check out the prerequisite lessons links above.

When dividing decimal numbers, we perform long division by ignoring the decimal points in both numbers, then placing the decimal point in the correct position in the quotient after everything is done.

  1. Do long division by ignoring the decimal points in both the dividend and divisor
  2. After calculating the quotient, move the decimal point to the left the same number of places as the decimal point is in the dividend, and move the decimal point to the right the same number of places as the decimal point is in the divisor
Let's calculate \(7.2 \div 9\) as an example. First, we set up long division and calculate the result by ignoring the decimal points in both numbers like so: $$ \begin{matrix} \ \ \ \ \ \ \ \ 8 \\ 9 \ \overline{)7.2} \\ - \ 7 \ 2 \\ \hline \ \ \ \ \ \ \ 0 \end{matrix} $$ Because the decimal point is 1 place from the right-most digit in the dividend 7.2, we need to move the decimal point 1 place to the left in the quotient. Because the decimal point is 0 places from the right-most digit in the divisor 9, we need to move the decimal point 0 places to the right in the quotient. $$ \begin{matrix} \ \ \ \ \ 0.8 \\ 9 \ \overline{)7.2} \\ - \ 7 \ 2 \\ \hline \ \ \ \ \ \ \ 0 \end{matrix} $$ The result is therefore \(7.2 \div 9=0.8\).

Exercise 3

Calculate \( 14.4 \div 1.2 \).


Show solution
The correct answer is \(12\).

$$ \begin{matrix} \ \ \ \ \ \ \ \ \ \ \ \ \ 1 \ 2 \\ \ \ \ 1.2 \ \overline{)14.4} \\ \ \ \ \ -12 \\ \hline \ \ \ \ \ \ \ \ \ \ \ \ 2 \ 4 \\ \ \ \ \ \ \ \ \ \ - 2 \ 4 \\ \hline \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ 0 \end{matrix} $$ By ignoring the decimal points and performing long division, we get 12 as the quotient. Because the decimal point is 1 place from the right-most digit in the dividend 14.4, we need to move the decimal point 1 place to the left in the quotient. Because the decimal point is 1 place from the right-most digit in the divisor 1.2, we need to move the decimal point 1 place to the right in the quotient. The result is therefore \(14.4 \div 1.2 = 12\).

Exercise 4

Calculate \( 4.08 \div 1.2 \).


Show solution
The correct answer is \(3.4\).

$$ \begin{matrix} \ \ \ \ \ \ \ \ \ \ \ \ \ \ 3.4 \\ \ \ \ 1.2 \ \overline{)4.0 \ 8} \\ \ \ \ \ \ -3 \ 6 \\ \hline \ \ \ \ \ \ \ \ \ \ \ \ \ \ 4 \ 8 \\ \ \ \ \ \ \ \ \ \ \ \ - 4 \ 8 \\ \hline \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ 0 \end{matrix} $$ By ignoring the decimal points and performing long division, we get 34 as the quotient. Because the decimal point is 2 places from the right-most digit in the dividend 4.08, we need to move the decimal point 2 places to the left in the quotient. Because the decimal point is 1 place from the right-most digit in the divisor 1.2, we need to move the decimal point 1 place to the right in the quotient. The result is therefore \( 4.08 \div 1.2 = 3.4\).

When dividing decimal numbers, sometimes it might seem like the dividend is too big and cannot be divided by the divisor. In cases like these, simply find the equivalent dividend with trailing zeroes added, and follow the same procedure above.

For example, we cannot directly calculate \(24 \ div 4.8\), but since \(24 = 24.0\), \(24 \ div 4.8 = 24.0 \div 4.8 \), which we can calculate.

Exercise 5

Calculate \( 24 \div 4.8 \).


Show solution
The correct answer is \(5\).

$$ \begin{matrix} \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ 5 \\ \ \ \ 4.8 \ \overline{)24.0} \\ \ \ \ \ \ \ \ \ -24.0\\ \hline \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ 0 \end{matrix} $$ \( 24 \div 4.8 = 24.0 \div 4.8\). By ignoring the decimal points and performing long division, we get 5 as the quotient. Because the decimal point is 1 places from the right-most digit in the dividend 24.0, we need to move the decimal point 1 place to the left in the quotient. Because the decimal point is 1 place from the right-most digit in the divisor 4.8, we need to move the decimal point 1 place to the right in the quotient. The result is therefore \( 24 \div 4.8 = 5\).
