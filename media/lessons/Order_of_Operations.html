Up until now, we have learned about the some elementary operations: addition (\(+\)), subtraction (\(-\)), multiplication (\(\times\)), division (\(\div\)), and exponents. We have learned how to solve simple problems such as \(2+5\) or \(8 \div 4\). However, many problems are more complicated and involve more than one operation, such as \(3 \times 2 - 5\) or \(8-4\div 2\). The question then becomes, in what order do we solve these equations with more than one operation? The order of operations is a set of rules that we follow to solve equations with more than one operation.

The order of operations is best remembered by the acronym PEMDAS, where each letter stands for the following operation:

  • P: Parentheses (\(()\), also known as brackets)
  • E: Exponents
  • M, D: Multiplication and Division
  • A, S: Addition and Subtraction
Precedence (what happens first and what happens later) is determined by the order of appearance in the acronym. In other words, the order in which we solve an equation is to first solve all operations inside parentheses, then solve all exponents, then solve all multiplication and division operations, and finally solve all addition and subtraction operations.

For example, let's try to solve \(3+5\times 2\). Because there are no parentheses or exponents, we can skip the first two stages. Because multiplication/division takes precedence over addition/subtraction, we first multiply then add. Therefore, \(3+5 \times 2 = 3 + 10 = 13\).

Exercise 1

Calculate \((3+5) \times 2\)


Show solution
The correct answer is \( 16 \).

Because there are parentheses, we must solve all operations inside the parentheses first. Therefore, \((3+5) \times 2 = 8 \times 2 = 16\).


Exercise 2

Calculate \(1 + 16 \div 2^3\)


Show solution
The correct answer is \( 3 \).

Exponent operations must be done before division, and division operations must be done before addition. Therefore, \(1 + 16 \div 2^3 = 1 + 16 \div 8 = 1 + 2 = 3\).


One important note to keep in mind is that multiplication and division have the same precedence (in other words, neither operation "overpowers" another), and addition/subtraction also have the same precedence. If multiplication/division or addition/subtraction happen to occur together, we simply do the operations from left to right.

Exercise 3

Calculate \(6 \div 2 \times 3\)


Show solution
The correct answer is \( 9 \).

Multiplication and division have the same precedence, so we do the operations from left to right. \(6 \div 2 \times 3 = 3 \times 3 = 9\).

Exercise 4

Calculate \(5 - 1 + 3 \times 2\)


Show solution
The correct answer is \( 10 \).

Multiplication happens before subtraction and addition, so we first get \(5 - 1 + 3 \times 2 = 5 - 1 + 6\). Then, because subtraction and addition have the same precedence, we go left to right to get \(5 - 1 + 6 = 4 + 6 = 10\).

Exercise 5

Calculate \(2^{1+2}+10 - (2+3) \times (2 + 1) \)


Show solution
The correct answer is \( 3 \).

First, we start by doing all operations inside parentheses to get \(2^{1+2}+10 - (2+3) \times (2 + 1) = 2^{1+2}+10 - 5 \times 3\). Then, we complete all operations involving exponents to get \(2^{1+2}+10 - 5 \times 3 = 2^3+10 - 5 \times 3 = 8+10-5 \times 3\). Then, we complete all operations involving multiplication/division to get \(8+10-5 \times 3 = 8 +10-15\). Finally, we complete all operations involving addition/subtraction to get \(8 +10-15 = 18-15=3\).
