Each and every object that we can see has an outline/form, and we call this their shape.
Because many shapes have similar properties, we can categorize these shapes based on their properties.

We can broadly categorize shapes into two categories: two-dimensional shapes and three-dimensional shapes.

A two-dimensional shape (or 2D shape) is any "flat" shape that does not have any thickness. For example, a piece of paper takes the form of a 2D shape because it is flat and has no thickness.
A three-dimensional shape (or 3D shape) is any shape that is not flat. In other words, it has some depth/thickness. For example, a soccer ball takes the form of a 3D shape.

Because three dimensional shapes are difficult to represent on a flat surface like the screen you are using right now, we sometimes use a combination of solid and dashed lines to show depth.
